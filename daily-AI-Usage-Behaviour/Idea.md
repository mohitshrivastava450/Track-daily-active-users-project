# Daily Active Users (DAU) Tracking with GenAI

## Project Overview
Build a data analytics platform with GenAI to track, analyze, and predict daily active users. Uses AI to provide insights about user engagement patterns.

## Key Objectives

1. **Real-Time Tracking** - Monitor DAU daily with live dashboards
2. **Predictive Analytics** - Forecast future DAU and identify churn risks
3. **GenAI Insights** - Ask questions like "What's our DAU trend?" and get AI answers
4. **Dashboards & Alerts** - Visual reports and automated alerts for anomalies
